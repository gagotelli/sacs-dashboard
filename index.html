About this dashboard

This is a reference dashboard for the SACS network, built from the SACS Topology (Current Final, with CNS) diagram: core switching, security/firewall, SAH and BBC access & distribution layers, VLANs, critical infrastructure services, and port/speed documentation.

Status is not live yet. Every device shows "Unknown" until a poller is wired up against the vendor APIs available on-site (Meraki, Palo Alto, ClearPass). See README.md in this repo for the integration plan and the exact file to populate.
